TypeScript

short introduction

What is TypeScript ?

It is a strict syntactical superset of JavaScript, and adds optional static typing to the language.

Who developed ?

microsoft

but it's open source

Why did they developed TS ?

while developing VSC they felt that they start more and more time spend on understanding and maintaining code than writing new one

Why we want TS ?

  • It scales
  • "Feature gap" not so big
  • Self documentation

  • Number
  • String

Boolean


							let value = true; // type boolean set by default
							let bool: boolean = false;

Number


							let dec: numder = 7;
							let hex: numder = 0xf0f;
							let bin: numder = 0b1011;
							let oct: numder = 0o723;

String


							let name: string = "Ivan";
							let city: string = 'Magadan';

Array


							let numbers: numder[] = [0, 1, 2, 3, 4];
							let words: string[] = ['word', 'word', 'word'];

Tuple


							let data: [number, string];
							data = [10, 'ten'];
							data = ['10', 'ten']; // Error
							data[3] = false; // Error, 'boolean' isn't 'number | string'
							data[3] = 'word';
							data[3] = 9;

Enum


							enum stringNumber {Zero, One, Two, Three, Four};
							let num: string = stringNumber[3];
							console.log(num); // Displays 'Three'

Any


							let data: any = true;
							data = 'string';
							data = 30;
							let anyArray: any[] = [true, 'word', 42];

Void


								let isUndefined: void = undefined;
								let isNull: void = null;

Functions


							function add(x: number, y: number): number {
								return x + y;
							}

Optional and Default Parameters


							function buildName(firstName: string, lastName?: string) {
								return lastName ? firstName + ' ' + lastName : firstName;
							}
							let res = buildName('Ivan'); // 'Ivan'
							res = buildName('Ivan', 'Petrov'); // 'Ivan Petrov'
						

							function buildName(firstName: string, lastName = 'Petrov') {
								return firstName + ' ' + lastName;
							}
							let res = buildName('Ivan'); // 'Ivan Petrov'

TS compiler

This is what compiles your TypeScript code in to JavaScript.

That's it or not ?

TSC options


								tsc sandbox.ts
Will compile in sandbox.js 

								tsc sandbox.ts --watch
Will recompile every time sandbox.ts changes 

								tsc --init
Initializes a TypeScript project and creates a tsconfig.json file

tsconfig.json

Allows you to config compiler 


								{
									"compilerOptions": {
										"target": "es5",
										"removeComments": true,
										"outFile": "../../built/local/tsc.js"
									},
									"include": [
										"src/**/*"
									],
									"exclude": [
										"node_modules",
										"**/*.spec.ts"
									]
								}
